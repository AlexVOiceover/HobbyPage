The Gaggiuino mod

Gaggiuino is an open-source modification for the Gaggia Classic and Pro expresso machines developed by Zer0-bit and a small community

Comparison

The Gaggia Classic is a simple but robust machine, perfect for modding.

This is how looks a modified Gaggia.

Procedure

This modification consists in adding pressure, temperature and weight sensor to the machine. Using relays, an Arduino board (newer versions use a STM32 Blackpill) will control the temperature and pressure transforming this humble machine in a great performing expresso machine

What we get with this mod:

  • Temp Control
  • Pressure Control
  • Weight Contol
  • Brew time Contol
